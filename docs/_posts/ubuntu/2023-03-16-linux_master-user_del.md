## $userdel

유저 계정을 삭제합니다.

| 옵션 | 내용                                                                                                                                     |
| ---- | ---------------------------------------------------------------------------------------------------------------------------------------- |
| -f   | 강제로 제거합니다. 사용자가 로그인했을 때,메일 스풀이 없을 때, 사용자와 동일한 이름의 그룹이 다른 유저의 Primary Group일때도 삭제합니다. |
| -r   | 홈디렉토리와 메일 스풀을 삭제합니다.                                                                                                     |
