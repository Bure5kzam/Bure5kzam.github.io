 코드 스페이스는 클라우드에서 편집기 환경을 구성해 접속시 사용할 수 있게 해준다.소스코드, 컴파일, 컴파일러, 디버거, 편집기, 사용자 도스 정의 파일을 구성해 접속시 사용할 수 있게 해준다.

- 코드 스페이스간 클라우드를 통한 설정 동기화 (setting sync)


- 사용자 설정은 깃허브 사용자 기본 설정에서 활성화 시킬 수 있다고함.
