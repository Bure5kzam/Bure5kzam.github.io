# 문자열 입력

int fgetc(*fp);
한글자씩 읽을 때 사용
EOF (-1)을 리턴 받으면 파일 포인터의 마지막을 나타냄

```c
#include <stdio.h>
int main() {
    FILE *fp = fopen()"tmp.txt", "r");
    for(int ch = fgets(fp); ch != EOF; ch = fgetc(fp)) {
        printf("%c", ch);
    }
    fclose(fp);
    return 0;
}
```
