# 질문
1. IPC란?
   1. 프로세스의 메모리는 커널에서 할당받은 유저모드 공간에 할당됨.
   2. 독립적인 프로세스들 간에 통신할수 있도록 해주는게 IPC
   3. 종류
      1. PIPE (반 이중 통신, read, write)
      2. Shared Memory
      3. Memory Map
