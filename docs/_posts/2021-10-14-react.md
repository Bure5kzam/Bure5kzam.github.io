# 개요
 기존 사용하던 Vue.js 대신 React 를 사용하기 위해 1주간의 학습기간을 가졌다.
 프로젝트 중간에는 도저히 시간이 안난다. 작업을 해도 해도 만족이 안되니 끝이 없는 것 같다. 오히려 마감 기한 덕분에 건강이 지켜지는 것 같아 고마울 정도다.
